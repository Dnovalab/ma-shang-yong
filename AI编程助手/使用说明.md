# AI编程助手 —— 使用说明

## 这是什么

一个桌面程序，打开就能用国产 AI（DeepSeek）写代码。不需要翻墙、不需要海外信用卡、不需要任何配置。

## 使用方法

### 第一步：下载

在 GitHub Releases 页面下载对应系统的版本：
- **Mac 用户**：下载 `.dmg` 文件
- **Windows 用户**：下载 `.exe` 文件

### 第二步：安装

**Mac：** 打开 dmg 文件 → 把 AI编程助手 拖到 Applications → 打开

**Windows：** 打开 exe 文件 → 按提示安装 → 打开

### 第三步：获取 API Key

打开后的程序会提示你设置 API Key。按以下步骤操作：

```
① 点右上角齿轮 ⚙️
② 点击"去 DeepSeek 注册"链接
③ 浏览器打开后，用手机号注册 DeepSeek 账号
④ 登录后进入 API Keys 页面
⑤ 创建一个 API Key，复制
⑥ 粘贴到程序的 API Key 输入框
⑦ 点"保存"
```

### 第四步：开始使用

在输入框输入编程问题，按回车发送。

## 示例问题

```
用 Python 写一个计算器程序
帮我解释这段 JavaScript 代码
怎么写一个 REST API 接口
把这段代码从 Python 改成 Java
帮我找找这段代码的 bug
```

## 常见问题

**Q: DeepSeek 收费吗？**
A: 注册送 ¥10 免费额度，够用很久。用完后 ¥0.5/百万 tokens，约等于几毛钱用一天。

**Q: 需要联网吗？**
A: 需要联网调用 API。但不需要翻墙，直连国内即可。

**Q: 我的对话记录会泄露吗？**
A: 所有对话记录只保存在你本地电脑上，不会上传到任何服务器。
