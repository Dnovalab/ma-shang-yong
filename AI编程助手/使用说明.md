# AI编程助手 —— 使用说明

> 打开就能用的国产 AI 编程工具
> 支持 DeepSeek、通义千问、智谱 GLM

---

## 下载安装

在 GitHub Releases 页面下载对应系统的版本：
- **Mac 用户**：下载 `.dmg` 文件，打开拖到 Applications 文件夹
- **Windows 用户**：下载 `.exe` 文件，按提示安装

---

## 配置 API Key（唯一需要手动操作的步骤）

打开程序后，点右上角齿轮 ⚙️ 进入设置：

**① 选择模型**

| 模型 | 厂商 | 适合场景 | 注册地址 |
|------|------|----------|----------|
| DeepSeek Chat | DeepSeek | 通用编程，性价比高 | platform.deepseek.com |
| DeepSeek Reasoner | DeepSeek | 复杂推理、debug | platform.deepseek.com |
| 通义千问 Max | 阿里云 | 中文理解强，长上下文 | dashscope.aliyun.com |
| 智谱 GLM-4 | 智谱 | 综合能力均衡 | open.bigmodel.cn |

**② 获取 API Key**

选好模型后，点击下方的注册链接去对应平台注册：
- DeepSeek：用手机号注册 → 进入 API Keys 页面 → 创建 Key → 复制
- 阿里云：用支付宝/手机号注册 → 进入 DashScope → API Keys 管理 → 创建
- 智谱：用手机号注册 → 进入 API Keys → 创建

**③ 粘贴并保存**

把复制的 API Key 粘贴到输入框，点「保存」即可开始使用。

> ⚠ **注意**：不同厂商的 API Key 不能混用。选 DeepSeek 模型就要用 DeepSeek 的 Key，选通义千问就要用阿里云的 Key。

---

## 开始使用

在输入框输入编程问题，按回车发送。

**示例：**
- 用 Python 写一个计算器程序
- 帮我解释这段 JavaScript 代码
- 怎么写一个 REST API 接口
- 把这段代码从 Python 改成 Java
- 帮我找找这段代码的 bug

## 常见问题

**Q: DeepSeek 收费吗？**
A: 注册送 ¥10 免费额度，够用很久。用完后 ¥0.5/百万 tokens。

**Q: 通义千问和智谱收费吗？**
A: 都有免费额度。阿里云新用户送 ¥200 体验金，智谱有免费调用次数。

**Q: 需要联网吗？**
A: 需要联网调用 API，但直连国内即可，不需要翻墙。

**Q: 对话记录安全吗？**
A: 所有记录只保存在你本地电脑上，不会上传到任何服务器。

**Q: 设置需要每次打开都配吗？**
A: 只需要配置一次，会自动保存。
